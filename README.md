## 실행
- 브라우저로 `index.html` 열기 → 즉시 동작
- 설치·인터넷 연결 불필요 (CSS·JS·SVG 모두 임베드)

## 5화면 흐름
1. **학교 선택** — 12개 표본 학교(학교알리미 실데이터 익명 추출) 중 선택
2. **AI 진단** — 에너지 등급(추정) + 또래군 대비 격차 + 과밀(실측)
3. **리모델링 시뮬** — 단열·창호·LED·냉난방을 토글, 효율 점수 실시간 변동
4. **결과 리포트** — BEFORE/AFTER + AI 자동 진단 편지(학부모용·교육청용)
5. **교육청 모드** — 효율안(229교, 1,031 GWh) vs 형평안(404교, 428 GWh), 취약학교 수혜 10%→49%

## 데이터 라벨
- **실측**: 과밀·학생수·학교급 (학교알리미 1,543교)
- **실측 분석값**: 229/404교·10→49% (전국 1,486곳 K4 형평 분석)
- **추정**: 학교별 에너지·찜통일수·냉난방비 (공개 데이터 부재 → 노후도 기반 추정. 화면에 항상 명시)
